# Voice AI Pac-Man

A browser-based Pac-Man game controlled by multilingual voice commands.

This project was developed as a university research project to investigate multilingual speech recognition for game interaction.

## Features

- Voice-controlled Pac-Man
- English support
- Japanese support
- Katakana pronunciation support
- Experimental mixed-language mode
- Real-time research data collection
- Score, levels and high score system
- Power pellet and frightened ghost mode
- Four ghost behaviors
- Full-screen mode

## Technologies

- HTML5
- CSS3
- JavaScript
- Web Speech API

## Future Improvements

- Whisper API integration
- OpenAI Realtime API integration
- True multilingual recognition
- Burmese support
- Chinese support
- Korean support

## Research Purpose

This project investigates how multilingual speech recognition can be used as an alternative game interaction method for browser-based games.
